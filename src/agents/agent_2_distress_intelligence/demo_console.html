📞 999 Emergency Call — Operator Form
How this works in production: A 999 operator receives a voice call. The caller says where they are and what's happening. The operator types the details into this form and submits. The form POSTs structured JSON to Agent 2's POST /ingest/hotline endpoint — the same endpoint a real 999 system integration would use.

For the demo: You play the role of the 999 operator. The caller's location comes from what they SAY on the phone — the operator transcribes it. No voice-to-text ML needed.
💬 SMS / USSD Message Sender
How this works in production: People send SMS to a shortcode (e.g., 16789). The Telco's SMS gateway receives the message and HTTP POSTs it to Agent 2's POST /ingest/sms endpoint. The NLP parser then extracts location, water level, and urgency from the text — whether it's structured format, USSD response, or free-text Banglish.

For the demo: This form simulates sending an SMS. The parsing logic that runs on the message is identical to what runs in production. SMS works on 2G — this is the critical channel when internet dies.

📱 Social Media Feed
How this works in production: Agent 2 connects to Twitter API v2 and Facebook Graph API, searches for flood-related posts in Dhaka using trilingual keywords (Bengali + English + Banglish). The NLP pipeline extracts zone, water level, urgency, and rescue need from each post.

Current status: Twitter API key not yet available. Using sample data from the existing sample_dataset_generator.py in Agent 1. The NLP parsing runs identically on both real and sample data.

To get Twitter API: Apply at developer.twitter.com. Free tier gives 500K tweets/month read access. Once approved, set TWITTER_BEARER_TOKEN in your .env.
🛰️ Satellite Flood Alerts (from Agent 1)
How this works: Agent 1 detects flooding via Sentinel-1 SAR imagery and publishes flood_alert messages to Redis. Agent 2 subscribes to this channel and feeds alerts into the Satellite+Population overlay engine, which combines flood extent with BBS census data to estimate population at risk.

This channel needs zero communication from victims. Even if all phone networks are down, satellite data + census overlay gives Agent 3 enough to pre-position rescue boats.
